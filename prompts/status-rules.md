# 状態管理ルール

Human Host Wikiでは、すべての情報に状態を付けます。

| 状態 | 使う条件 |
| --- | --- |
| 確認済み | 自分でゲーム内確認した情報 |
| 未確認 | 情報はあるが、自分では確認していない情報 |
| 開発中 | 開発者が制作中と示している情報 |
| 実装予定 | 将来追加予定として示されている情報 |
| 変更可能性あり | アーリーアクセスのため変わる可能性が高い情報 |
| 不明 | 判断できない情報 |

## Codexへの指示例

この情報を整理してください。実装済み、未確認、開発中、実装予定を分けてください。未確認情報を断定しないでください。
