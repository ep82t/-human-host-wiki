## 用語管理ルール

- 公式名が分かるまでは仮称として扱います。
- 英語名が未確認の場合は「未確認」と書きます。
- 翻訳では、用語集の表記を優先します。
- 7 Days to Die由来の説明は参考として使い、Human Hostの仕様として断定しません。
