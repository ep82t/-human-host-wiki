# 用語集

Human Host Wikiで使う用語、日本語名、英語名、表記ゆれ、状態ラベルを管理します。  
将来的な英語版対応のため、早い段階から日英対応を残します。

## 状態ラベル

| 日本語 | 英語 | 意味 |
| --- | --- | --- |
| 確認済み | Confirmed | ゲーム内で確認済み |
| 未確認 | Unconfirmed | 情報はあるがゲーム内未確認 |
| 開発中 | In Development | 開発中とされている |
| 実装予定 | Planned | 将来追加予定 |
| 変更可能性あり | Subject to Change | アーリーアクセスのため変わる可能性あり |
| 不明 | Unknown | 判断材料が不足 |

## 用語管理ルール

- 公式名が分かるまでは仮称として扱います。
- 英語名が未確認の場合は「未確認」と書きます。
- 翻訳では、用語集の表記を優先します。
- Steamコミュニティガイド由来の名称は、ゲーム内確認まで未確認にします。
- 7 Days to Die由来の説明は参考として使い、Human Hostの仕様として断定しません。
