# アップデート履歴

Human Hostの更新内容と、Wiki内で影響を受けるページを管理します。  
アーリーアクセス中のため、アップデート後は古い情報をそのまま信じず、確認待ちに戻すことがあります。

## 監視する情報源

| 情報源 | 用途 | 優先度 |
| --- | --- | --- |
| Steam公式ニュース | パッチ、告知、公式説明 | 高 |
| Steam公式ストア | 大枠の仕様、タグ、説明文 | 高 |
| Steamコミュニティガイド | 個別調査、検証メモ | 中 |
| 開発者情報 | 開発中・実装予定の追跡 | 高 |
| ゲーム内確認 | 最終確認 | 最高 |

## アップデート一覧

| 日付 | 状態 | 内容 | 影響ページ |
| --- | --- | --- | --- |
| 2026-06-20 | 記録 | Steam公式・コミュニティガイドを出典として整理開始 | バイオーム、資源、クラフト、建築、敵、車両 |
