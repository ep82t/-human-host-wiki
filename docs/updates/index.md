# アップデート履歴

Human Hostの更新内容と、Wiki内で影響を受けるページを管理します。

## 運用ルール

- 公式情報、開発者情報、ゲーム内確認を分けて記録します。
- 変更された可能性があるページをメモします。
- 古くなった情報には「変更可能性あり」または「要再確認」を付けます。
