# 資源一覧

クラフト、建築、武器、車両運用に関係する資源を整理します。

## 管理項目

| 項目 | 内容 |
| --- | --- |
| 状態 | 確認済み / 未確認 / 開発中 / 実装予定 / 不明 |
| 入手方法 | 採取、分解、探索、取引など |
| 入手場所 | バイオーム、施設、コンテナなど |
| 用途 | クラフト、建築、修理など |
| 重要度 | 序盤・中盤・終盤での重要度 |
| 確認日 | 最後に確認した日 |

## 資源一覧
