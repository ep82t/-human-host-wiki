# 敵一覧

敵の出現場所、行動、危険度、ドロップを整理します。

## 管理項目

| 項目 | 内容 |
| --- | --- |
| 状態 | 確認済み / 未確認 / 開発中 / 実装予定 / 不明 |
| 出現場所 | バイオーム、施設、時間帯など |
| 行動 | 移動、攻撃、索敵など |
| 危険度 | 主観的な危険度 |
| ドロップ | 倒した時に得られるもの |
| 対策 | 武器、立ち回り、拠点防衛 |
