## 概要

砂漠は、Steamコミュニティガイドで `biome5.砂漠/Desert` として紹介されているバイオームです。  
地面表層から砂が多く採れるとされています。

## 資源

| 区分 | 資源 |
| --- | --- |
| 表層 | Sand / 砂 |
| 岩盤層 | Nitrate Ore / 硝酸塩鉱石、Yellow Wax Stone / おうろう石 |
| 特徴的なもの | Cactus / サボテン |
| 固有アイテム | Solar Panel / ソーラーパネル、High-Energy Laser Emitter / 高エネルギー激光発射器、Circular Saw / 丸鋸 |

## 探索メモ

- 固有アイテム候補が多く、探索価値が高そうです。
- 砂の大量採取場所として確認したいバイオームです。
