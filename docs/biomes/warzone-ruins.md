## 概要

戦域・廃墟は、Steamコミュニティガイドで `biome3.戦域(廃墟)/WarZone(Ruins)` として紹介されているバイオームです。  
建築・工業系素材を集めやすい可能性があります。

## 資源

| 区分 | 資源 |
| --- | --- |
| 表層 | Clay / 粘土 |
| 岩盤層 | Iron Ore / 鉄鉱石、Epidote / りょくれんせき |
| その他 | Lime / 石灰 |
| 特徴的なもの | Rebar / 鉄筋、Scrap Iron / 廃鉄、Cement / 水泥 |
| 固有アイテム | Magnet / 磁石、Bearing / ベアリング、Pulley / 滑車 |

## 探索メモ

- ガレキを壊すと、鉄筋、セメント、水泥、廃鉄、廃黄銅などが手に入るとされています。
- 廃墟系素材の確認優先度が高いバイオームです。
