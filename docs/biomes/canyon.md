## 概要

峡谷は、Steamコミュニティガイドで `biome6.峡谷/Canyon` として紹介されているバイオームです。  
砂漠に近い資源構成を持ちつつ、原木も特徴的なものとして挙げられています。

## 資源

| 区分 | 資源 |
| --- | --- |
| 表層 | Sand / 砂 |
| 岩盤層 | Nitrate Ore / 硝酸塩鉱石、Yellow Wax Stone / おうろう石 |
| 特徴的なもの | Cactus / サボテン、Log / 原木 |
| 固有アイテム | Solar Panel / ソーラーパネル、High-Energy Laser Emitter / 高エネルギー激光発射器、Circular Saw / 丸鋸 |

## 探索メモ

- 砂漠と共通する固有アイテムが挙げられています。
- 砂漠との差分として、原木の入手確認が重要です。

## 未確認事項

- 砂漠との差分
- 原木の入手条件
- 車両移動のしやすさ
- 地形による危険度
