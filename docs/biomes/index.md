# バイオーム一覧

Human Hostのバイオーム情報を整理するページです。  
資源分布、敵の出現、探索しやすさ、拠点構築への向き不向きを記録します。

## 調査状況

| 項目 | 状態 |
| --- | --- |
| 公開Web検索 | 信頼できるバイオーム一覧は未発見 |
| ゲーム内確認 | 未着手 |
| スクリーンショット確認 | 未着手 |
| 開発者情報 | 未整理 |

現時点では、バイオーム名を断定できる情報が不足しています。  
このページでは、ゲーム内確認または信頼できる開発者情報が得られたものから追加します。


## 確認するとよいこと

- バイオーム名がゲーム内で表示されるか
- 見た目で分類した仮称なのか、正式名称なのか
- 資源の種類が他バイオームと違うか
- 敵の出現がバイオーム依存か
- 車両移動に向いているか
- 拠点を建てやすい地形か
- 温度、毒、放射線などの環境ダメージがあるか
- アップデートで地形や資源分布が変わったか
