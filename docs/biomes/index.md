# バイオーム一覧

Human Hostのバイオーム情報を整理するページです。  
資源分布、敵の出現、探索しやすさ、拠点構築への向き不向きを記録します。

## 管理項目

| 項目 | 内容 |
| --- | --- |
| 状態 | 確認済み / 未確認 / 開発中 / 実装予定 / 不明 |
| 主な資源 | 採取できる資源 |
| 敵 | 出現する敵 |
| 探索ポイント | 建物、施設、目印など |
| 拠点向き | 拠点構築に向くか |
| 危険度 | 主観的な危険度 |
| 確認日 | 最後に確認した日 |
