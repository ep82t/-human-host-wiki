## 情報の状態

Human Hostはアーリーアクセス中のため、情報は次の状態で管理します。

| 状態 | 意味 |
| --- | --- |
| 確認済み | ゲーム内で確認した情報 |
| 未確認 | 情報はあるが、まだ確認していない情報 |
| 開発中 | 開発者情報などで制作中とされている情報 |
| 実装予定 | 将来追加予定として扱う情報 |
| 変更可能性あり | アーリーアクセスのため変わる可能性が高い情報 |
| 不明 | 現時点では判断できない情報 |

## 運用方針

| 項目 | 方針 |
| --- | --- |
| 主目的 | 自分が見やすい形で情報を整理する |
| 対象 | Human Hostのサバイバルクラフト情報 |
| 記事形式 | Markdown |
| データ形式 | YAML |
| AI利用 | Codexで整理、下書き、翻訳、更新候補抽出を行う |
| 英語版 | 将来対応予定 |
| 自動更新 | 将来対応予定 |

## 注意点

- 架空のゲーム内データは追加しません。
- 未確認情報は未確認として明記します。
- Trader Systemは開発中として扱い、実装済みとは書きません。
- 数値、ドロップ率、出現条件などは確認できるまで断定しません。
- 7 Days to Dieとの類似は参考として扱い、Human Hostの仕様として断定しません。
