# Human Host Wiki

Human Hostの情報を個人で見やすく整理するためのWikiです。  
アーリーアクセス中のサバイバルクラフトゲームとして、バイオーム、資源、アイテム、クラフト、建築、敵、車両、アップデート履歴、未実装要素を中心に管理します。

## 現在の運用方針

| 項目 | 方針 |
| --- | --- |
| 主目的 | 自分が見やすい形で情報を整理する |
| 対象 | Human Hostのサバイバルクラフト情報 |
| 記事形式 | Markdown |
| データ形式 | YAML |
| AI利用 | Codexで整理、下書き、翻訳、更新候補抽出を行う |
| 英語版 | 将来対応予定 |
| 自動更新 | 将来対応予定 |

## 情報の状態

| 状態 | 意味 |
| --- | --- |
| 確認済み | ゲーム内で確認した情報 |
| 未確認 | 情報はあるが、まだ確認していない情報 |
| 開発中 | 開発者情報などで制作中とされている情報 |
| 実装予定 | 将来追加予定として扱う情報 |
| 変更可能性あり | アーリーアクセスのため変わる可能性が高い情報 |
| 不明 | 現時点では判断できない情報 |

- Human Hostは発売直後のアーリーアクセスゲームとして扱います。
