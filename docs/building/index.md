# 建築一覧

Human Hostの建築、拠点、防衛設備、生産設備、移動拠点に関係する情報を整理します。

## 調査状況

| 項目 | 状態 | 根拠 |
| --- | --- | --- |
| 建築自由度 | 公式記載あり | Steam公式ストア |
| 破壊可能な地形・建物 | 公式記載あり | Steam公式ストア |
| 移動拠点 | 公式記載あり | Steam公式ストア |
| 具体的な建築レシピ | 未確認 | ゲーム内確認待ち |
| 耐久値・修理 | 未確認 | ゲーム内確認待ち |

## 建築カテゴリ

| 分類 | 状態 | 内容 | 確認すること |
| --- | --- | --- | --- |
| 基本ブロック | 未確認 | 壁、床、屋根など | 種類、必要素材、耐久 |
| 防衛設備 | 未確認 | 罠、防壁、侵入対策 | 起動条件、修理、味方判定 |
| 収納 | 未確認 | アイテム保管 | 容量、設置条件 |
| 生産設備 | 未確認 | クラフトや加工に関係する設備 | 作業台、炉、電力 |
| 拠点設備 | 未確認 | 生活、防衛、移動に関係する設備 | 電力、照明、修理 |
| 移動拠点 | 未確認 | 車両と建築を組み合わせた拠点 | 安定性、操作、燃料 |

## 建築メモ

- Steam公式ストアでは、地形、建物、木、岩などが破壊可能と説明されています。
- 建築素材を組み合わせて移動拠点を作れることが説明されています。
- 具体的な建築レシピや必要素材は、まだゲーム内確認待ちです。

## 確認するとよいこと

- 建築UIの開き方
- 建築パーツのカテゴリ
- 必要素材
- 建築物の耐久
- 敵が建築物を攻撃するか
- 修理素材
- 回収や移動ができるか
- 車両に建築物を載せられるか
