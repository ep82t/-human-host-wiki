# 未確認情報

まだゲーム内で確認していない情報、開発者情報、推測、再確認が必要な情報を管理します。

## 未確認情報の扱い

| 種類 | 扱い |
| --- | --- |
| 自分で未確認 | 未確認として記録 |
| 開発者情報 | 情報源を明記して記録 |
| 他者情報 | 情報源と確認状況を明記 |
| 推測 | 推測と明記し、断定しない |
| 古い情報 | 要再確認として扱う |

## 未確認リスト

| 項目 | 分類 | 状態 | 情報源 | 確認すること |
| --- | --- | --- | --- | --- |
| Trader System | システム | 開発中 | 開発者情報 | 実装後の仕様確認 |

## 記録テンプレート

```md
## 項目名

- 分類:
- 状態:
- 情報源:
- 記録日:
- 確認したいこと:
- 関連ページ:
- メモ:
```
