# 未確認情報

まだゲーム内で確認していない情報、開発者情報、Steamコミュニティガイド由来の情報、推測、再確認が必要な情報を管理します。

## 未確認情報の扱い

| 種類 | 扱い |
| --- | --- |
| Steam公式ストア | 大枠の仕様として扱う。個別数値は確認待ち |
| Steamコミュニティガイド | 参考情報として採用。ゲーム内確認までは未確認 |
| 開発者情報 | 情報源を明記し、開発中・実装予定として扱う |
| 自分でゲーム内確認 | 確認済みに変更 |
| 推測 | 推測と明記し、断定しない |
| 古い情報 | 要再確認として扱う |

## 未確認リスト

| 項目 | 分類 | 状態 | 情報源 | 確認すること |
| --- | --- | --- | --- | --- |
| バイオーム別資源 | バイオーム・資源 | 未確認 | Steamコミュニティガイド | ゲーム内で採取できるか |
| 固有アイテム候補 | アイテム | 未確認 | Steamコミュニティガイド | 入手方法と用途 |
| 素材加工 | クラフト | 未確認 | 調査待ち | 作業台、レシピ、加工量 |
| 建築クラフト | 建築 | 未確認 | Steam公式ストア | 具体的な建築パーツと必要素材 |
| 車両組立 | 車両 | 未確認 | Steam公式ストア、Steamガイド | 部品、燃料、修理、Lua制御 |
| 敵・夜の群れ・ボス | 敵 | 未確認 | Steam公式ストア | 出現条件、ドロップ、対策 |
| Trader System | システム | 開発中 | 開発者情報 | 実装後の仕様確認 |

## 確認済みに変える条件

- 自分でゲーム内確認した
- スクリーンショットまたは動画で確認できる
- 日付とバージョンを記録できる
- 情報源をページに残せる

## 記録テンプレート

```md
## 項目名

- 分類:
- 状態:
- 情報源:
- 記録日:
- 確認したいこと:
- 関連ページ:
- メモ:
```
