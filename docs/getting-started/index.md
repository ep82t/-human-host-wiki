## 初心者向けの方針

- まず生存を優先する。
- 食料、水、医薬品の確保を最初に確認する。
- 夜や敵の行動が分かるまでは無理に遠出しない。
- 拠点候補と帰れる場所を決めてから探索する。
- 情報が未確認の場合は、断定せずメモとして残す。
