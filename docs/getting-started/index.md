## 初回プレイの考え方

- 最初は設定を大きく変えず、標準に近い状態で始める
- キャラクターは目的に合わせて選ぶ
- ゲーム開始後はF1で手作りメニューを確認する
- 地下に小さな仮拠点を作り、寝袋・収納箱・木工作業台を置く
- 未確認情報は断定せず、確認できたら更新する
