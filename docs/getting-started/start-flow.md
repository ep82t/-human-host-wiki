# ゲーム開始までの流れ

Human Hostには現時点でチュートリアルがないため、ゲーム開始前の画面で何を選ぶのかを先に把握しておくと迷いにくくなります。

開始までの流れは、現在確認できている範囲では次の順番です。

1. メインメニューで「ニューゲーム」を選ぶ
2. セーブスロットを選ぶ
3. 難易度・ワールド設定を確認する
4. キャラクターを選ぶ
5. ゲームを開始する

:::note
公式Steamページでは、キャラクター特性やサバイバルクラフト要素は確認できますが、開始手順の細かい順番までは説明されていません。  
このページはゲーム内スクリーンショットで確認した内容をもとに整理しています。
:::

### 初心者向けの見方

| 見る場所 | まずどうするか |
| --- | --- |
| 難易度プリセット | 初回はノーマルのままでよい |
| 戦利品スポーン | 物資量を変える設定。初回はそのままでよい |
| 死亡時ドロップ | 死んだ時に落とすもの。厳しすぎる場合だけ変更候補 |
| ゾンビ設定 | 夜や群れの危険度に関係する。初回は大きく変えない |
| ワールドシード | ワールド生成に関係する。最初は気にしなくてよい |
| 昼夜サイクル | 1日の長さ。慣れるまでは標準でよい |

詳しい設定項目は、あとで [ニューゲーム設定](/settings/new-game/) に整理します。

## 4. キャラクターを選ぶ

設定を進めるとキャラクター選択になります。  
キャラクターごとにスキルと初期装備が違うため、最初の遊びやすさに影響します。

| 迷った時の目的 | おすすめ |
| --- | --- |
| とにかく安定して始めたい | スカベンジャー |
| 戦闘が不安 | 狩人 |
| 車両クラフトを早く試したい | 整備士 |

詳しくは [キャラクター選択](/getting-started/character-selection/) を見てください。

## 5. ゲーム開始後にやること

ゲームが始まったら、まずF1で手作りメニューを開き、序盤に必要な道具と地下仮拠点を作ります。
