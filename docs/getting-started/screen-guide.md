# 画面の見方

Human Hostの画面表示、メニュー、HUD、状態表示を整理するページです。  
現時点ではオープニング画面のスクリーンショットのみ確認済みです。

![Human Host opening menu](/images/screenshots/opening-menu.png)

## オープニング画面

スクリーンショットから確認できる項目:

| 表示 | 状態 | メモ |
| --- | --- | --- |
| シングルプレイ | 確認済み | メニューに表示 |
| マルチプレイ | 確認済み | メニューに表示。ただしSteam公式ではCo-op開発中の記載あり |
| ワークショップ | 確認済み | メニューに表示 |
| ゲーム設定 | 確認済み | メニューに表示 |
| ゲームを終了 | 確認済み | メニューに表示 |

## HUDで確認したいこと

- 体力
- 空腹
- 水分
- スタミナ
- 状態異常
- 装備スロット
- クラフト通知
- 時刻や昼夜
- バイオーム名表示

## 未確認事項

- ゲーム内HUDの正式な表示内容
- 状態異常の種類
- 空腹・水分がどのように表示されるか
- スキルや成長画面があるか
- マップや座標表示があるか
