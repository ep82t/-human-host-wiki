# 最初に集めたい資源

序盤は、いきなり遠くへ探索するよりも、地下仮拠点を作るための素材を先に集めると安定します。

## 最優先の素材

| 素材 | 主な使い道 | 集め方 |
| --- | --- | --- |
| 原木 | 簡易な石の稿、木材系クラフト | 木を切る、木材を壊す |
| 石 | 簡易な石の稿、石系ツール | 石を拾う、岩や地形から採取 |
| 植物繊維 | 麻ロープ | 草のある場所で採取 |
| 麻ロープ | 簡易な石の稿など | 植物繊維 x2 からクラフト |
| 破損の木板 | 地下拠点の入り口ふさぎ、簡易建築 | 初期装備や探索で入手 |

## 植物繊維

草がある場所で、**Ctrlを押してしゃがみ、マウス右ボタンを押しっぱなしにしながら左クリック**すると採取できます。

## 麻ロープ

麻ロープは植物繊維から作れます。序盤では「簡易な石の稿」の材料として必要です。

<a className="hh-wide-shot hh-small-shot" href={useBaseUrl('/images/screenshots/early-game/hemp-rope.png')} target="_blank" rel="noreferrer">
  <img src={useBaseUrl('/images/screenshots/early-game/hemp-rope.png')} alt="麻ロープ" />
</a>

| 完成品 | 必要素材 |
| --- | --- |
| 麻ロープ x1 | 植物繊維 x2 |
