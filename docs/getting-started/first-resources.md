# 最初に集めたい資源

序盤に優先して確認したい資源を整理します。  
現時点ではSteamコミュニティガイド由来の未確認情報をもとにしています。

## 優先資源

| 優先度 | 資源 | 入手候補 | 理由 |
| --- | --- | --- | --- |
| 高 | 粘土 | 多くのバイオーム | 表層資源として広く出る候補 |
| 高 | 原木 | 荒廃地・廃墟、峡谷 | 建築・燃料候補 |
| 高 | 鉄鉱石 | 戦域・廃墟 | 金属素材候補 |
| 中 | 銅鉱石 | 山地、森林 | 序盤金属素材候補 |
| 中 | 植物繊維 | 熱帯雨林 | 基本クラフト素材候補 |
| 中 | キノコ | 森林 | 食料候補 |
| 中 | 水泥 | 戦域・廃墟 | 建築素材候補 |

## 序盤探索の候補地

| 場所 | 狙い | 注意 |
| --- | --- | --- |
| 山地 | 粘土、銅鉱石、大理石 | 初期スポーン候補だがゲーム内確認待ち |
| 森林 | 粘土、銅鉱石、大理石、キノコ | 道が入り組む可能性 |
| 戦域・廃墟 | 鉄鉱石、鉄筋、廃鉄、水泥 | 敵や危険度は未確認 |

## 確認するとよいこと

- 最初に採れる資源
- 必要ツール
- どの資源がクラフトに直結するか
- 拠点素材として使える資源
- 食料や水の確保とどちらを優先すべきか
