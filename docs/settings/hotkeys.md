# ホットキー

ゲーム設定の「ホットキー」画面で確認できた項目です。  
以下はスクリーンショットから読み取れた内容のため、現在のWikiでは「確認済み」として扱います。

## マウス

| 項目 | 設定値 | 状態 |
| --- | --- | --- |
| マウス感度 | 100% | 確認済み |
| スコープ感度 | 300% | 確認済み |
| マウスY軸反転 | オフに見える | 確認済み |

## 注意点

- 画面右下に「すべての設定はゲーム開始後に有効になります。」と表示されています。
- スクリーンショットに写っていないホットキーは未記録です。
- 今後のアップデートで初期キーが変わる可能性があります。
