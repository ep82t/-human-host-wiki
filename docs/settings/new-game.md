# ニューゲーム設定

ニューゲーム開始前に表示される設定画面の項目を整理します。  
公式Steamページでは、Human Hostがランダム生成ワールド、昼夜、ゾンビの群れ、建築破壊を含むサバイバルクラフトゲームであることが説明されています。個別の設定項目の詳細説明は公式ページ上では確認できないため、このページでは「スクリーンショットで確認できた内容」と「Wiki向けの説明」を分けて扱います。

## 基本方針

| 区分 | 扱い |
| --- | --- |
| 項目名と設定値 | スクリーンショットで確認できたものは「確認済み」 |
| 項目の意味 | 公式情報と一般的なゲーム設定からの説明。ツールチップ画像で後から更新 |
| おすすめ設定 | まだ未作成。実プレイ確認後に追加 |

## 開始までの流れ

| 手順 | 内容 | 状態 |
| --- | --- | --- |
| 1 | メニューから「ニューゲーム」を選ぶ | 確認済み |
| 2 | 空のセーブスロットを選ぶ | 確認済み |
| 3 | 難易度やワールド設定を確認する | 確認済み |
| 4 | 「次へ」を押して開始手順を進める | 確認済み |

## 初心者向けの見方

| 目的 | 見る項目 |
| --- | --- |
| 最初は標準で遊びたい | 難易度プリセットをノーマルのままにする |
| 探索の報酬量を変えたい | 戦利品スポーン、戦利品リスポーン間隔 |
| 死んだ時の厳しさを変えたい | 死亡時ドロップ |
| 夜の危険度を変えたい | ゾンビ走行モード、昼夜サイクルの長さ、昼の長さ |
| 拠点防衛の厳しさを変えたい | ゾンビのブロックへのダメージ、群集ゾンビ数 |
| PC負荷や画面上の死体量を調整したい | 最大屍体数、屍体消滅時間 |

## 今後確認すること

- 各項目にカーソルを置いた時のゲーム内説明文。

- 「激怒」「狂乱」「突然変異レベル」の具体的な効果。
- 群集パラメータが通常ゾンビとどう違うか。
- 初心者向けのおすすめ設定。
