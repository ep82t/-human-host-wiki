# ニューゲーム設定

ニューゲーム開始前に表示される設定画面の項目を整理します。  
公式Steamページでは、Human Hostがランダム生成ワールド、昼夜、ゾンビの群れ、建築破壊を含むサバイバルクラフトゲームであることが説明されています。個別の設定項目の詳細説明は公式ページ上では確認できないため、このページでは「画像で確認できた内容」と「Wiki向けの説明」を分けて扱います。

## 基本方針

| 区分 | 扱い |
| --- | --- |
| 項目名と設定値 | 画像で確認できたものだけを掲載 |
| 項目の意味 | 公式情報と一般的なゲーム設定からの説明。ツールチップ画像で後から更新 |
| おすすめ設定 | まだ未作成。実プレイ確認後に追加 |

## セーブスロット

| 表示 | 説明 |
| --- | --- |
| 自動保存 | 自動保存用の枠。画像ではAとして表示 |
| 1から5 | 手動または新規ゲーム用のセーブスロット |
| 空 | 新規ゲームに使える空きスロット |
| 決定 | 選択したスロットで進む |
| 削除 | 選択したセーブデータを削除する操作と思われる |

## 難易度

| 項目 | 値 | Wiki向け説明 |
| --- | --- | --- |
| 難易度プリセット | ノーマル | 複数の設定をまとめて標準的な難易度にするプリセットと思われる |

## 戦利品パラメータ

| 項目 | 値 | Wiki向け説明 |
| --- | --- | --- |
| 戦利品スポーン | 100% | 探索で見つかる戦利品の出現量に関係する設定と思われる |

## プレイヤーパラメータ

| 項目 | 値 | Wiki向け説明 |

## ゾンビ移動パラメータ

| 項目 | 値 | Wiki向け説明 |
| --- | --- | --- |
| ゾンビ走行モード | 夜間のみ走行 | 公式ページでは夜にゾンビの群れが走ることが説明されている。夜だけ走る標準設定と思われる |
| ゾンビ激怒 | 無効化 | ゾンビの一時的な強化や行動変化に関係する設定と思われる |
| ゾンビ狂乱 | 無効化 | ゾンビのより危険な行動変化に関係する設定と思われる |

## 初心者向けの見方

| 目的 | 見る項目 |
| --- | --- |
| 最初は標準で遊びたい | 難易度プリセットをノーマルのままにする |
| 探索の報酬量を変えたい | 戦利品スポーン、戦利品リスポーン間隔 |
| 死んだ時の厳しさを変えたい | 死亡時ドロップ |
| 夜の危険度を変えたい | ゾンビ走行モード、昼夜サイクルの長さ、昼の長さ |
| 拠点防衛の厳しさを変えたい | ゾンビのブロックへのダメージ、群集ゾンビ数 |
| PC負荷や画面上の死体量を調整したい | 最大屍体数、屍体消滅時間 |

## 今後確認すること

- 各項目にカーソルを置いた時のゲーム内説明文。
- プリセットを変更した時、どの項目がどう変わるか。
- 「激怒」「狂乱」「突然変異レベル」の具体的な効果。
- 群集パラメータが通常ゾンビとどう違うか。
- 初心者向けのおすすめ設定。
