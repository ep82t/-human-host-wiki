# ゲーム設定

Human Hostの設定画面で確認できた項目を整理します。  
このページは、スクリーンショットで確認できた内容だけを「確認済み」として扱います。

## 設定カテゴリ

| カテゴリ | 状態 | ページ |
| --- | --- | --- |
| ホットキー | 確認済み | [ホットキー](/settings/hotkeys/) |
| グラフィックス | 未確認 | 今後追加 |
| ゲーム | 未確認 | 今後追加 |
| オーディオ | 未確認 | 今後追加 |
| 言語 | 未確認 | 今後追加 |

## 運用メモ

- スクリーンショットで見えている項目だけを記録する。
- スクロールで見切れている項目は、確認できるまで追加しない。
- 初期キー設定なのか、変更後の設定なのか不明な場合はメモに残す。
- アップデートで設定項目やキー割り当てが変わる可能性がある。
